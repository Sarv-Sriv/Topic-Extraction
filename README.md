# Topic-Extraction
This projects helps to find the important topic from the PDFs

I made this project to save time when writing long reports and project documentation.
It uses a local LLM (Ollama) to generate text only from the PDF you upload, which helps reduce AI hallucinations.

Since everything runs locally, your research and data stay private and are not uploaded online.

The generated text scores between 18% and 72% AI detection, with the lowest around 10%.
The results were tested using QuillBot, GPTZero, and Grammarly.
